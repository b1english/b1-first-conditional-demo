### 题2｜语法9876题

- 功能：通过条件块和结果块拼出 if 条件句，强化语法结构。
- 语法卡英文：If ..., ... will ...
- 顶部语法卡中文：如果（某个条件发生），就会（产生某个结果）。
- 完整句子：If we take a bus, we will save money.
- 题干：请翻译：
- 选项/组块：we take a bus / will save
- 句子大块：如果我们坐公交车：If(如果) __we take a bus__(我们坐公交车),
- 句子大块：我们就会省钱：we(我们) __will save__(就会省) money(钱).
- 完整句查词单位：If / we / take a bus / we / will / save / money
- 语法词汇讲解：If ..., ... will ... = 如果（条件发生），就会（随之产生某个结果）。
if 后面说条件，用一般现在时。
后面说结果，用 will + 动词原形。
